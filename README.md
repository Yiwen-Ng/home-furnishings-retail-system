## Features
- User registration and authentication
- Product browsing, search, filter, and sorting
- Shopping cart and order management
- 3D product preview for furniture visualization
- Online payment integration using third-party payment API
- Admin dashboard for order and inventory management
- Wishlist and review system

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- Payment API
- 3D Preview

## How to Run Locally
1. Install XAMPP
2. Import database from `database/home_furnishings.sql`
3. Place project folder in `htdocs`
4. Update database configuration
5. Run via `localhost/project-name`

## Disclaimer
Payment API keys and credentials are excluded for security purposes.
